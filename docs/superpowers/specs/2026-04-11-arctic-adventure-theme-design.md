# Arctic Adventure Visual Theme — Design Spec

See full specification at: `Documentation/SUMMARACE_VISUAL_THEME.md`

This spec was brainstormed and approved on 2026-04-11. Key decisions:
- Theme: Skyblue Arctic Day
- Every 2D scene uses 7-layer backgrounds (sky → mountains → trees → snow → particles → foreground → UI)
- All UI uses Cartoon UI plugin sprites (816 available)
- Kenney Nature Kit for environment (1640+ assets)
- Easy Chara "Hisa Teacher" = Ms. Lumi character
- Per-scene accent colors
- DOTween recommended for animations
