## ⚠️ Important Reminders

1. **Always check the license** before publishing on Play Store. Most free assets allow commercial use, but some require attribution.

2. **Create a `CREDITS.txt`** in your project listing every asset and creator. Show this in the game's About screen. Example format:
   ```
   SummaRace — Asset Credits
   
   3D Environment: Free Snow Mountain by ProAssets (Unity Asset Store)
   3D Trees: Low Poly Trees Pack Lite (Unity Asset Store)
   Player Character: Mixamo (Adobe)
   Sound Effects: Dustyroom Free Casual Game SFX Pack (CC0)
   Music: [Track Name] by [Artist] (Pixabay)
   Fonts: Fredoka, Patrick Hand (Google Fonts, Open Font License)
   ```

3. **Don't mix art styles.** Pick assets that look like they belong together. Use Kenney + Mixamo (consistent low-poly) OR Synty (consistent stylized) — not both.

4. **Compress textures** when importing. Set Max Size to 1024 for mobile to keep build size small.

5. **Backup your assets** in cloud storage (Google Drive, Dropbox) before importing into Unity. If something breaks, you don't want to re-download everything.

6. **Check the Unity Asset Store free filter regularly:** https://assetstore.unity.com/?free=true — new free assets are added every month!
